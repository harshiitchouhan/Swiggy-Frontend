# 🍔 Swiggy Clone

A modern and responsive Swiggy-inspired food ordering web application built with React. The project replicates the core user experience of Swiggy, including restaurant browsing, menu exploration, search functionality, cart management, and dynamic API-driven content.

## 🚀 Live Demo

🔗 https://your-live-link.vercel.app

---

## ✨ Features

- 🍽 Browse restaurants dynamically
- 🔍 Search restaurants and dishes
- 📋 View detailed restaurant menus
- 🛒 Add and remove items from cart
- 📱 Fully responsive design
- ⚡ Fast performance with React
- 🔄 Real-time API data integration
- 🎨 Clean and modern UI

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/yourusername/swiggy-clone.git
```

Navigate to project directory:

```bash
cd swiggy-clone
```

Install dependencies:

```bash
npm install
```

Start development server:

```bash
npm run dev
```

---

## 🌐 Environment Setup

If you are using a backend proxy for API requests:

```env
VITE_API_URL=http://localhost:4000
```
